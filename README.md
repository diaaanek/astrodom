## 📑 Installation Guide
To get started, have the backend API running
` `
Fork and clone repo:
` https://github.com/diaaanek/ `

Install all dependencies:

`   `

Create database scheme with migrations:

` rake db:migrate `

Seed database:

` rake db:seed `

Start local server:

` rails s `
